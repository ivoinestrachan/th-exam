— THE RULE —
Every bag returns empty, zipped and clipped. Every tool lives inside the toolbox.
Always
  • Empty bags fully
  • Close zips, clip buckles
  • Keep tools in the toolbox
  • Stack with zips facing out
Never
  • Store full or stuffed bags
  • Leave tools loose on shelves
  • Mix tools with bags
  • Store damp or smelly items
If a shelf is overfilled → remove until it closes cleanly. If tools are on a bag shelf → return to the toolbox now. If a bag smells → air-dry or wash before storing.
