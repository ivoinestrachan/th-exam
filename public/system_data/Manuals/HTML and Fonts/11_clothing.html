— THE RULE —
Every garment lives inside a fabric box. Every needle and pin lives inside a tin or case. Nothing loose.
Always
  • Fold and box all clothing
  • Case needles and pins
  • Fold or roll fabric
  • Wash before storing
Never
  • Leave clothing loose
  • Mix clothes with sewing
  • Overfill the boxes
  • Store damp or dirty items
If a box is overfilled → redistribute into a second box. If clothing slips outside its box → refold and return. If something damp was stored → remove, dry, clean the box.
