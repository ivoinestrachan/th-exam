— THE RULE —
Every item returns to its assigned shelf, immediately after use.
Always
  • Return to assigned shelf
  • Check expiry dates
  • Fold neatly first
  • Keep first aid grabbable
Never
  • Mix categories
  • Store damp towels
  • Leave expired meds
  • Overstuff bags
If a towel is damp → hang it to dry first, then fold and store. If medicine is expired → remove and dispose at a pharmacy. If bags slump → empty, refold, restack.
