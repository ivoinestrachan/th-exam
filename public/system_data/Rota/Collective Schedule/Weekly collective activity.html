Daily group calisthenics. 10 min warm-up, 20 min jog, then circuit.

Cleaning
Work through each room on the day's list using the room manual. Clean rooms top to bottom. Tick off only when done to standard.
S
Supplies
Walk each listed room with its supply list. Restock and log anything missing on the inventory sheet. Alert if bulk supply low.

Cooking
Cook the meal from the meal set, then clean the kitchen thoroughly afterwards within the slot.

Procurement
Create basket, place orders, pick up orders.
L
Laundry
Two loads daily. Morning is always Mixed; evening is the labelled type. Wash, dry, fold, return.
R
Reset
Lightly tidy and restore each room to order per the room manual. Set up the sleeping room ready for the night.
